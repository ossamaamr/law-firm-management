# دليل البدء السريع - CasEngine\n\nابدأ مع نظام إدارة مكتب المحاماة في 5 دقائق فقط!\n\n## 📦 المتطلبات الأساسية\n\n- Node.js 18+ ([تحميل](https://nodejs.org/))\n- Git ([تحميل](https://git-scm.com/))\n- PostgreSQL ([تحميل](https://www.postgresql.org/)) أو استخدم Manus\n\n## 🚀 البدء السريع\n\n### 1️⃣ استنساخ المستودع\n\n```bash\ngit clone https://github.com/ossamaamr/law-firm-management.git\ncd law-firm-management\n```\n\n### 2️⃣ تثبيت الاعتماديات\n\n```bash\nnpm install -g pnpm\npnpm install\n```\n\n### 3️⃣ إعداد البيئة\n\n```bash\ncp .env.example .env\n```\n\nعدّل `.env` وأضف:\n```env\nDATABASE_URL=postgresql://user:password@localhost:5432/law_firm\nJWT_SECRET=your-secret-key-here\nNODE_ENV=development\n```\n\n### 4️⃣ إعداد قاعدة البيانات\n\n```bash\npnpm db:push\n```\n\n### 5️⃣ تشغيل التطبيق\n\n```bash\npnpm dev\n```\n\n✅ التطبيق متاح على `http://localhost:3000`\n\n## 🔐 حساب اختبار\n\nاستخدم هذه البيانات للدخول:\n\n```\nمعرّف المكتب: @testfirm#\nاسم المستخدم: admin\nكلمة المرور: TestPassword123!\n```\n\n## 📁 هيكل المشروع\n\n```\n.\n├── client/              # تطبيق React\n│   └── src/\n│       ├── pages/       # الصفحات الرئيسية\n│       ├── components/  # المكونات\n│       └── lib/         # المكتبات\n├── server/              # خادم Express\n│   ├── routers.ts       # مسارات tRPC\n│   └── db.ts            # قاعدة البيانات\n├── drizzle/             # مخطط قاعدة البيانات\n└── docs/                # التوثيق\n```\n\n## 🛠️ الأوامر المهمة\n\n### التطوير\n```bash\npnpm dev              # تشغيل بيئة التطوير\npnpm build            # بناء التطبيق\npnpm test             # تشغيل الاختبارات\npnpm test:watch      # تشغيل الاختبارات المراقبة\n```\n\n### قاعدة البيانات\n```bash\npnpm db:push         # تطبيق الهجرات\npnpm db:studio       # فتح واجهة قاعدة البيانات\npnpm db:seed         # ملء البيانات الأولية\n```\n\n### الإنتاج\n```bash\npnpm build           # بناء للإنتاج\nNODE_ENV=production node dist/server.js\n```\n\n## 🎨 تخصيص التطبيق\n\n### تغيير الألوان\n\nعدّل `client/src/index.css`:\n\n```css\n@layer base {\n  :root {\n    --primary: 59 130 246;      /* الأزرق */\n    --secondary: 107 114 128;   /* الرمادي */\n    --accent: 34 197 94;        /* الأخضر */\n  }\n}\n```\n\n### تغيير اللغة\n\nعدّل `client/src/contexts/LanguageContext.tsx`:\n\n```typescript\nconst [language, setLanguage] = useState<'ar' | 'en'>('ar');\n```\n\n## 🔐 الأمان\n\n### متغيرات البيئة الحساسة\n\n**لا تضع أبداً في الكود:**\n- `JWT_SECRET`\n- `DATABASE_URL`\n- `API_KEYS`\n\n**استخدم `.env` بدلاً من ذلك:**\n```bash\n# .env\nJWT_SECRET=your-secret-key\nDATABASE_URL=postgresql://...\n```\n\n## 🐛 استكشاف الأخطاء\n\n### المشكلة: \"Connection refused\"\n\n```bash\n# تحقق من قاعدة البيانات\nsudo systemctl status postgresql\n\n# أعد التشغيل\nsudo systemctl restart postgresql\n```\n\n### المشكلة: \"Port already in use\"\n\n```bash\n# ابحث عن العملية\nlsof -i :3000\n\n# أوقفها\nkill -9 <PID>\n```\n\n### المشكلة: \"Module not found\"\n\n```bash\n# أعد تثبيت الاعتماديات\nrm -rf node_modules pnpm-lock.yaml\npnpm install\n```\n\n## 📚 الموارد المفيدة\n\n- [وثائق React](https://react.dev/)\n- [وثائق Express](https://expressjs.com/)\n- [وثائق Tailwind](https://tailwindcss.com/)\n- [وثائق Drizzle](https://orm.drizzle.team/)\n- [وثائق tRPC](https://trpc.io/)\n\n## 🤝 المساهمة\n\n1.
