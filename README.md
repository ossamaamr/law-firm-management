# CasEngine - نظام إدارة مكتب المحاماة

نظام إدارة متكامل وشامل لمكاتب المحاماة يوفر جميع الأدوات اللازمة لإدارة القضايا والموكلين والفواتير والموارد البشرية.

## 🚀 البدء السريع

### 1. استنساخ المستودع
```bash
git clone https://github.com/ossamaamr/law-firm-management.git
cd law-firm-management
```

### 2. تثبيت الاعتماديات
```bash
pnpm install
```

### 3. إعداد متغيرات البيئة
```bash
cp .env.example .env
```

### 4. إعداد قاعدة البيانات
```bash
pnpm db:push
```

### 5. تشغيل التطبيق
```bash
pnpm dev
```

التطبيق سيكون متاحاً على `http://localhost:3000`

## 🔐 الأمان

### ميزات الأمان المطبقة

1. **تشفير كلمات المرور**
   - استخدام bcrypt مع salt rounds = 12
   - تحقق من قوة كلمة المرور

2. **معدل التحديد (Rate Limiting)**
   - حماية من هجمات Brute Force
   - حد أقصى 5 محاولات دخول في 15 دقيقة

3. **تنظيف المدخلات (Input Sanitization)**
   - إزالة الأحرف الخاصة والخطرة
   - التحقق من صحة البيانات

4. **التشفير**
   - تشفير البيانات الحساسة
   - استخدام AES-256

5. **JWT Tokens**
   - توقيع آمن للجلسات
   - انتهاء الصلاحية التلقائي

## 📊 الأداء

### تحسينات الأداء

1. **التخزين المؤقت (Caching)**
   - تخزين مؤقت في الذاكرة مع TTL
   - تخزين مؤقت للاستعلامات

2. **معالجة الدفعات (Batch Processing)**
   - معالجة آمنة للعمليات الكبيرة
   - معالجة متزامنة محدودة

3. **مراقبة الأداء**
   - قياس وقت تنفيذ العمليات
   - تتبع استخدام الذاكرة
