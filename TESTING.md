# دليل الاختبار - CasEngine\n\nدليل شامل لاختبار نظام إدارة مكتب المحاماة.\n\n## 📋 أنواع الاختبارات\n\n### 1. اختبارات الوحدة (Unit Tests)\nاختبار الوحدات الفردية من الكود\n\n```bash\npnpm test -- --testPathPattern=\"unit\"\n```\n\n### 2. اختبارات التكامل (Integration Tests)\nاختبار تفاعل المكونات مع بعضها\n\n```bash\npnpm test -- --testPathPattern=\"integration\"\n```\n\n### 3. اختبارات الأمان (Security Tests)\nاختبار الثغرات الأمنية والحماية\n\n```bash\npnpm test -- security.test.ts\n```\n\n### 4. اختبارات الأداء (Performance Tests)\nاختبار سرعة وكفاءة النظام\n\n```bash\npnpm test -- cache.test.ts\n```\n\n### 5. اختبارات النهاية إلى النهاية (E2E Tests)\nاختبار سير العمل الكامل\n\n```bash\npnpm test:e2e\n```\n\n## 🧪 كتابة الاختبارات\n\n### مثال: اختبار وحدة\n\n```typescript\nimport { describe, it, expect, beforeEach } from 'vitest';\nimport { validateEmail } from '@/lib/validators';\n\ndescribe('Email Validation', () => {\n  it('should validate correct email', () => {\n    expect(validateEmail('test@example.com')).toBe(true);\n  });\n\n  it('should reject invalid email', () => {\n    expect(validateEmail('invalid-email')).toBe(false);\n  });\n\n  it('should reject empty email', () => {\n    expect(validateEmail('')).toBe(false);\n  });\n});\n```\n\n### مثال: اختبار تكامل\n\n```typescript\nimport { describe, it, expect, beforeEach, afterEach } from 'vitest';\nimport { createCase, getCase, updateCase } from '@/server/db';\n\ndescribe('Case Management', () => {\n  let caseId: string;\n\n  beforeEach(async () => {\n    // Setup: Create a test case\n    const result = await createCase({\n      caseNumber: 'TEST-001',\n      title: 'Test Case',\n      status: 'open',\n    });\n    caseId = result.id;\n  });\n\n  afterEach(async () => {\n    // Cleanup: Delete test case\n    await deleteCase(caseId);\n  });\n\n  it('should create and retrieve case', async () => {\n    const retrieved = await getCase(caseId);\n    expect(retrieved.title).toBe('Test Case');\n  });\n\n  it('should update case', async () => {\n    await updateCase(caseId, { status: 'closed' });\n    const updated = await getCase(caseId);\n    expect(updated.status).toBe('closed');\n  });\n});\n```\n\n### مثال: اختبار أمان\n\n```typescript\nimport { describe, it, expect } from 'vitest';\nimport { hashPassword, verifyPassword } from '@/server/security';\n\ndescribe('Password Security', () => {\n  it('should hash password securely', async () => {\n    const password = 'SecurePassword123!';\n    const hash = await hashPassword(password);\n    expect(hash).not.toBe(password);\n  });\n\n  it('should verify correct password', async () => {\n    const password = 'SecurePassword123!';\n    const hash = await hashPassword(password);\n    const isValid = await verifyPassword(password, hash);\n    expect(isValid).toBe(true);\n  });\n\n  it('should reject incorrect password', async () => {\n    const password = 'SecurePassword123!';\n    const
